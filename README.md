## How to use venv
- Create a Virtual Environment
```
python3 -m venv venv
```
- Activate the Virtual Environment
```
source venv/bin/activate
```

- Install Dependencies
```
pip install -r requirements.txt
```

- Verify Installation
```
pip list
```

- Deactivate the Virtual Environment
```
deactivate
```
